# BIPA-analitic-Dasboard-T3
Sistema Inteligente de Analítica Bibliotecaria desarrollado con GEnIA


## Descripción General

BIPA Analitic Dashboard T3 es una aplicación web interactiva diseñada para analizar el comportamiento de uso dela BIPA mediante el estudio de préstamos, usuarios y clasificaciones temáticas LCC.

El sistema permite explorar información, identificar patrones de consulta, detectar tendencias de uso y apoyar la toma de decisiones relacionadas con el desarrollo de colecciones bibliográficas.

## Problemática

La Biblioteca Interactiva Pedro Arrupe SJ. generan grandes volúmenes de datos relacionados con préstamos, usuarios y colecciones documentales. Sin embargo, en muchas ocasiones esta información no se utiliza estratégicamente para apoyar la toma de decisiones.

## Justificación

La analítica de datos permite identificar patrones de uso, tendencias de consulta y áreas temáticas de interés para fortalecer el desarrollo de colecciones.

## Objetivo

Analizar el comportamiento de uso de la colección bibliográfica mediante el estudio de préstamos, usuarios y clasificaciones LCC.

## Datos utilizados

- Items Información bibliográfica de la colecció
- Prestamos Registros históricos de circulación
- Usuarios Información de usuarios de la biblioteca
- Clasificación LCC Catálogo temático basado en Library of Congress Classification.

## KPIs

- Total de préstamos
- Usuarios activos
- Títulos consultados
- Promedio de préstamos por usuario

## Visualizaciones

- Top clasificaciones más prestadas
- Top carreras con mayor uso
- Top títulos más prestados
- Distribución por categoría

## Hallazgos

- Carrera con mayor uso
- Clasificación más consultada
- Títulos más utilizados

## Módulo avanzado

Análisis de tendencias de préstamos mediante series temporales.
